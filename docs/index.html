Why agents need this

No LLM Required

Pure regex and heuristics. Works offline, air-gapped, on CI. Zero inference cost. Your agent's LLM reasons over the output — decompose handles preprocessing.

Deterministic

Same input, same output, every time. No temperature, no sampling, no drift. Agents can depend on the schema.

Fast

11ms for 3 documents. 934 chars/ms throughput. The bottleneck is your agent's reasoning, not decompose.

Composable

JSON in, JSON out. Pipe it, chain it, embed it. Works with any model — GLM-5, Llama, Claude, GPT. MCP native.

One line to install

Add to your agent's MCP config. Works with Claude Code, OpenClaw, Cursor, and any MCP-compatible agent.

{
  "mcpServers": {
    "decompose": {
      "command": "uvx",
      "args": ["decompose", "--serve"]
    }
  }
}
